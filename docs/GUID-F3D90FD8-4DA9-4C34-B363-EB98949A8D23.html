2.2.4 Adding Harmony Modules to Project

  1. Go to MCC Project Graph window, as opened in previous step
  2. Add BSP Component. On Device Resources, navigate to Board Support Packages (BSPs) and click on + sign next to PIC32CXMTSH Development Board BSP

  3. Add Core Component

    1. On Device Resources, navigate to Core and click on + sign next to Core
    2. Click No when prompted for adding FreeRTOS component. This example is a Baremetal project

  4. Add Time Component

    1. On Device Resources, navigate to System Services and click on + sign next to Time
    2. Right-Click on TMR dependency and select Satisfiers → TC0 (tc0)
    3. Click on TC0 component, on Configuration Options set Clock Source to MCK0/128 on Channel 0 so configuration matches the following:

  5. Add G3 MAC RT Component

    1. On Device Resources, navigate to SmartEnergy → Drivers and click on + sign next to G3 MAC RT
    2. Right-Click on SPI dependency and select Satisfiers → FLEXCOM1 (flexcom1)
    3. FLEXCOM1 component is auto-configured when connected to G3 MAC RT component, as seen on the figure:
    4. Click on G3 MAC RT component, on Configuration Options set Pins and PLC Coupling so configuration matches the following:
    5. Right-Click on PCOUP dependency and select Satisfiers → PLC PHY Coupling (srv_pcoup)

  6. Add PLC PVDD Monitor Component

    1. On Device Resources, navigate to SmartEnergy → Services and click on + sign next to PLC PVDD Monitor
    2. Right-Click on ADC dependency and select Satisfiers → ADC (adc)
    3. Click on ADC component, on Configuration Options set Prescaler and Channel 1 Options so configuration matches the following:
    4. Click on PLC PVDD Monitor component, on Configuration Options set Channel 1 so configuration matches the following:
  7. Add TRNG Component. On Device Resources, navigate to Peripherals and click on + sign next to TRNG

  8. Optional. Add QSPI Memory Support if application requires using it

    1. On Device Resources, navigate to Drivers → I2C EEPROM and click on + sign next to MEMORY
    2. Right-Click on MEMORY dependency and select Satisfiers → SST26 (drv_sst26)
    3. Right-Click on SPI dependency and select Satisfiers → QSPI (qspi)

    4. These components are Autoconfigured, once added to the project they are ready to use

  9. Optional. Add SLCDC Display Support if application requires using it

    1. On Device Resources, navigate to Graphics → Driver and click on + sign next to SLCDC Driver
    2. Right-Click on Segmented Display dependency and select Satisfiers → SLCDC PIC32CXMT

    3. These components are Autoconfigured, once added to the project they are ready to use

  10. At this point all components are added and configured in the project graph
