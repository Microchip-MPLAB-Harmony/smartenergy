1.3.1.5 APP_ENERGY Module

The APP_ENERGY module is in charge of saving the data coming from the metrology to a file system in the external flash memory.

The main function of the module is APP_ENERGY_Task which runs a state-machine to control the module APP_ENERGY.

The state-machine waits for the Datalog module to be ready before running. Then, it updates the information related to tamper detection, initialize the real-time clock and recovers or initialize the different data information from the file system if it exists (Time-Of-Use, Energy and Demand values).

When the APP_ENERGY module is ready, it checks the tamper detection periodically and waits for data in the input queue.

If a special request for historical data is required, the state-machine obtain the data from the datalog and send it using the related callback.
