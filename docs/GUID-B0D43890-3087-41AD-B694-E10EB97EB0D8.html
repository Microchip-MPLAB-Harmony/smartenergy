1.1.3.4.27 DRV_METROLOGY_RMS_SIGN Enum

C

typedef enum
{
    /* Positive Value */
    RMS_SIGN_POSITIVE = 0,

    /*  Negative Value */
    RMS_SIGN_NEGATIVE = 1,

} DRV_METROLOGY_RMS_SIGN;

Summary

Describes the sign of some metrology measurements.

Description

This quality only affects to some RMS measurements.
