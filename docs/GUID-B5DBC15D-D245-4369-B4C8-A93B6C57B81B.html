1.2.7.4 Library Interface

RF PHY Sniffer library provides the following interfaces:

Functions

NameDescription
SRV_RSNIFFER_SerialRxMessageSerializes a received RF frame along with its parameters
SRV_RSNIFFER_SetTxMessageGives a transmitted RF TX request object to sniffer library so it is stored for later serialization
SRV_RSNIFFER_SerialCfmMessageSerializes a transmitted RF frame along with its parameters
