2.2.5 Clock Configuration

  1. Start Clock Manager from Plugins drop-down menu

  2. The application demo can work with default clock configuration

    1. Verify that PLLACK0 is 20000000 Hz

    2. Verify that PLLACK1 is 200000000 Hz

    3. Verify that PLLABCK is 240000000 Hz

    4. Verify that MCK0 is 200000000 Hz

    5. Verify that MCK1 is 240000000 Hz

Note: This setting is valid for PIC32CXMTSH devices; other devices will have different clock settings. For more information, refer the device datasheet.
