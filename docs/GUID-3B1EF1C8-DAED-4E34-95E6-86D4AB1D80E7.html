Harmonic Analysis Command

When the application code receives a “HRR[n]” command, it configures the metrology library to enable the harmonics computation and to calculate the data corresponding to the harmonic number n. Then, the application receives the data from the library and computes the rms currents and voltages of the selected harmonic. Example:

HRR[3]
The calculated harmonic Irms/Vrms:
Irms_Har_A(A)  Irms_Har_B(A)  Irms_Har_C(A)
2.353          1.958          1.998
Vrms_Har_A(V)  Vrms_Har_B(V)  Vrms_Har_C(V)
8.094          9.455          7.968
