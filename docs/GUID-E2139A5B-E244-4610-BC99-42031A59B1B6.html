2.2 Create your first PLC Application using PIC32CXMTSH-DB and PL460-EK

This tutorial demonstrates how to create a PLC application project with MPLAB® Harmony G3-PLC Driver and MPLAB® Harmony v3 modules, using MPLABX IDE and MPLAB® Code Configurator.

The tutorial implements a simple PLC application on a Microchip 32-bit microcontroller development board. This will guide you through the steps to create a new PLC application.

Note : This guide demonstrates the creation of a PLC demo on the PIC32CXMTSH-DB board. These steps can be extended for other Microchip hardware development boards supported in Harmony v3, by applying the device/board specific configuration changes.
