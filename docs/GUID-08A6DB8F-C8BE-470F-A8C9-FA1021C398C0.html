1.2.1.4.2 Library Interface (PRIME)

If the PLC Driver uses PRIME PHY, the PLC PHY Coupling service library provides the following interfaces:

Functions

NameDescription
SRV_PCOUP_Get_Default_ChannelGet the default PRIME channel
SRV_PCOUP_Get_Channel_ConfigGet the PLC PHY Coupling parameters for the specified PRIME channel
SRV_PCOUP_Set_Channel_ConfigSet the PLC PHY Coupling parameters for the specified PRIME channel

Data types and constants

NameTypeDescription
SRV_PLC_PCOUP_CHANNEL_DATAStructPLC PHY Coupling data
